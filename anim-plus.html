Description du projet

c’est une application qui permet au personnel de pointer les présences des participants et enregistrer les dossiers d'inscription. Elle convient à tous types d’activités : Accueil de Loisirs Sans Hébergement ; Accueil Périscolaire ; Ateliers. Cette application s’adresse également aux différents types de structures : ALSH ; Restauration scolaire ; Centre de loisirs ; Centre social ; Garderie périscolaire. Anim+ est une application simple qui permet de savoir à tout moment qui est présent dans la structure et consulter les dossiers d'inscription.
